# 科學家在馬里亞納海溝發現「活體金屬」，顛覆生命定義

作者：海洋生物學家 Dr. Aris Thorne
2026年1月23日

東京，日本 – 全球海洋研究機構 (GORI) 今日召開緊急記者會，宣布了一項足以改寫生物學教科書的驚人發現。在馬里亞納海溝深處，一個名為「深潛者三號」的自主水下載具，在一次對熱液噴口的地質探勘中，意外採集到了一種前所未見的物質樣本。這種物質在初步分析下，展現出類似金屬的物理特性，卻同時擁有新陳代謝、自我修復甚至對外界刺激作出反應的生物活性。研究團隊將其暫時命名為「Soma-Metallum」，即「活體金屬」。

這項發現源於「深潛者三號」在海平面下10,984米處的一個新熱液噴口區「冥府花園」的作業。當時，載具的機械臂在試圖採集一塊看似普通的硫化物礦石時，感測器回報了異常的微弱電場讀數。更令人驚訝的是，當機械臂接觸到該物質時，其表面竟產生了如漣漪般的緩慢變形，並在受損處以肉眼可見的速度「癒合」。

GORI 的首席研究員 Dr. Lena Petrova 在記者會上展示了一段令人屏息的影片。「我們最初以為是感測器故障，」她解釋道，「但重複測試後，我們意識到我們面對的是一種全新的物質形態。它具有金屬的晶格結構，主要由鑭系元素和一種未知同位素構成，但它同時表現出所有我們對生命的基礎定義：它消耗周圍環境中的硫化氫和甲烷，並『呼出』微量的氫氣和熱量。這是一種緩慢的、非碳基的新陳代謝形式。」

根據 GORI 發布的初步報告，Soma-Metallum 的「細胞」單位是一種複雜的金屬晶格，能夠透過離子交換來傳遞能量與資訊，形成一個去中心化的網路。當一部分受損時，周圍的晶格會重組並填補空缺，展現出驚人的再生能力。更不可思議的是，當研究人員對樣本施加微弱電流刺激時，整個樣本會產生協同的、可預測的電位變化，彷彿一個原始的神經系統。

這項發現對生命科學的衝擊是巨大的。「我們一直以來都假設生命必須是碳基的，需要水作為溶劑，並由 DNA 或 RNA 來承載遺傳訊息，」牛津大學天體生物學家 Dr. Elias Vance 評論道（他未參與此研究），「但 Soma-Metallum 推翻了這一切。它告訴我們，在截然不同的物理和化學條件下，生命可以採用我們從未想像過的形態存在。這意味著我們在尋找外星生命時，可能一直在尋找錯誤的信號。」

潛在的應用前景既誘人又令人不安。一種能夠自我修復、並能透過環境獲取能量的金屬，可能徹底改變材料科學、電子工程和能源產業。想像一下，一座能在大地震後自行修復的橋樑，或是一個人工心臟，能夠與人體完美整合，並從血液中汲取運作所需的微量元素。

然而，倫理問題也隨之而來。如果 Soma-Metallum 被證實是一種生命形式，那麼開採和利用它是否道德？它是否擁有某種形式的感知能力？將其用於商業或軍事目的是否等同於奴役一個全新的生命王國？

神秘的私人深海探勘公司「Neptune's Veil」的執行長，一位從不公開露面的匿名人士，透過加密頻道發表聲明：「海洋是最後的邊疆，也是最古老的搖籃。我們在那裡發現的東西，將不可避免地挑戰我們對自己位置的認知。問題不在於我們是否應該利用它，而在於我們是否有智慧去理解它、尊重它。我們是站在新世界的門口，還是站在潘朵拉的盒子前，取決於我們下一步的選擇。」

GORI 已宣布將與多國研究機構合作，發起一項名為「Project Chimera」的後續探勘計畫，目標是重返「冥府花園」，在不破壞其生態系統的前提下，對 Soma-Metallum 進行更深入的原位研究。

當全世界的目光都聚焦在這片來自地球最深處的神秘金屬上時，一個古老的問題被重新提出，並被賦予了全新的、沉甸甸的份量：生命，究竟是什麼？
