# 行為準則 (Code of Conduct)

## 我們的承諾

我們作為成員、貢獻者和領導者，承諾讓參與我們社群的經驗對每個人來說都是無騷擾的，無論年齡、體型、可見或不可見的殘疾、種族、性別特徵、性別認同和表達、經驗水平、教育程度、社會經濟地位、國籍、外貌、種族、宗教或性取向如何。

我們承諾以有助於建立開放、熱情、多元、包容和健康社群的方式行動和互動。

## 我們的標準

有助於為我們的社群創造積極環境的行為範例包括：

* 對他人展現同理心和友善
* 尊重不同的意見、觀點和經驗
* 給予並優雅地接受建設性的回饋
* 承擔責任並向受我們錯誤影響的人道歉，並從經驗中學習
* 不僅關注我們個人的利益，也關注整體社群的利益

不可接受的行為範例包括：

* 使用性化語言或圖像，以及任何形式的性關注或挑逗
* 挑釁 (Trolling)、侮辱或貶損的評論，以及人身或政治攻擊
* 公開或私下的騷擾
* 未經他人明確許可，發布他人的私人資訊，例如物理地址或電子郵件地址
* 在專業環境中可能被合理認為不適當的其他行為

## 執行責任

社群領導者負責澄清和執行我們可接受行為的標準，並將對他們認為不適當、威脅、冒犯或有害的任何行為採取適當和公平的糾正措施。

社群領導者有權利和責任刪除、編輯或拒絕不符合本行為準則的評論、提交 (commits)、程式碼、wiki 編輯、問題 (issues) 和其他貢獻，並在適當時溝通審核決定的原因。

## 範圍

本行為準則適用於所有社群空間，也適用於個人在公共空間正式代表社群時。
代表我們社群的例子包括使用官方電子郵件地址、透過官方社群媒體帳戶發布訊息，或在線上或線下活動中擔任指定代表。

## 執行準則

社群領導者在確定違反本行為準則的任何行動的後果時，將遵循這些社群影響準則：

### 1. 糾正 (Correction)

**社群影響**：使用不適當的語言或其他被認為在社群中不專業或不受歡迎的行為。

**後果**：社群領導者的私人書面警告，澄清違規的性質並解釋為何該行為不適當。可能會要求公開道歉。

### 2. 警告 (Warning)

**社群影響**：透過單一事件或一系列行動造成的違規。

**後果**：警告，如果繼續該行為將會有後果。在指定的時間內，不得與相關人員互動，包括不得主動與執行行為準則的人員互動。這包括避免在社群空間以及外部管道（如社群媒體）進行互動。違反這些條款可能會導致暫時或永久封禁。

### 3. 暫時封禁 (Temporary Ban)

**社群影響**：嚴重違反社群標準，包括持續的不適當行為。

**後果**：在指定的時間內，暫時禁止與社群進行任何形式的互動或公開交流。在此期間，不得與相關人員進行公開或私下互動，包括不得主動與執行行為準則的人員互動。違反這些條款可能會導致永久封禁。

### 4. 永久封禁 (Permanent Ban)

**社群影響**：表現出一種違反社群標準的模式，包括持續的不適當行為、騷擾個人，或對某些群體的個人進行攻擊或貶低。

**後果**：永久禁止在社群內進行任何形式的公開互動。
