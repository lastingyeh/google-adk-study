# Typescript 安裝指南
> 更新日期：2026 年 1 月 4 日

## 安裝 ADK 與 ADK 開發者工具

`npm install @google/adk @google/adk-devtools`

---

### 重點說明

*   **@google/adk**: Google Ads 開發工具包 (ADK) 的核心套件，提供與 Google Ads API 互動所需的主要功能。
*   **@google/adk-devtools**: ADK 的開發者工具，包含用於簡化開發流程的輔助工具，例如程式碼產生器和設定驗證。
