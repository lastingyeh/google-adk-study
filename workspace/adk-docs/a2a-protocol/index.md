# 使用 Agent2Agent (A2A) 協定的 ADK
🔔 `更新日期：2026-01-15`

[`ADK 支援`: `Python` | `Go` | `Experimental`]

透過 Agent Development Kit (ADK)，您可以構建複雜的多代理系統，其中不同的代理需要使用 [Agent2Agent (A2A) 協定](https://a2a-protocol.org/) 進行協作和互動！本章節提供了一個全面的指南，用於構建強大的多代理系統，使代理能夠安全且高效地進行溝通與協作。

瀏覽下方的指南以了解 ADK 的 A2A 功能：

  **[A2A 簡介](./intro.md)**

  從這裡開始，透過構建一個包含根代理（root agent）、本地子代理（local sub-agent）和遠端 A2A 代理的多代理系統，來學習 A2A 的基礎知識。以下指南介紹了如何公開您的代理，以便其他代理可以透過 A2A 協定使用它：

  *   **[Python A2A 快速入門（公開）](./a2a-quickstart%20(exposing)/quickstart-exposing.md)**
  *   **[Go A2A 快速入門（公開）](./a2a-quickstart%20(exposing)/quickstart-exposing-go.md)**

  這些指南向您展示了如何讓您的代理使用 A2A 協定來調用另一個遠端代理：

  *   **[Python A2A 快速入門（調用）](./a2a-quickstart%20(consuming)/quickstart-consuming.md)**
  *   **[Go A2A 快速入門（調用）](./a2a-quickstart%20(consuming)/quickstart-consuming-go.md)**
