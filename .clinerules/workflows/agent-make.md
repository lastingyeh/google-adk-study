## 目標
根據 `python/agents` 目錄下的使用者提供檔案名稱，建立符合範本的目錄結構和相關檔案。

## 執行步驟
1. 分析agents目錄下所有實作專案
2. 根據專案設計列出agent功能與特色，根據 `docs/agents/agents.html` js 區塊 `agentsData` 內容進行比較
3. 插入資料至 `docs/agents/agents.html` 的 `agentsData` 陣列中
4. 確保新增的資料格式與現有資料一致，並包含必要的欄位如名稱、描述、功能等
5. 儲存並驗證 `docs/agents/agents.html` 檔案的正確性
6. 完成後，回報新增的 agent 資料內容
