# Google ADK 學習路線圖

## 概述

此儲存庫包含學習 Google ADK（代理開發工具包）的完整路線圖。此路線圖旨在引導學習者掌握有效使用 Google ADK 構建智能代理所需的重要主題和技能。


### 🕗 Day 1
<details>

<summary>

</summary>
